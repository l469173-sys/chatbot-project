# Chatbot Project（聊天機器人專題）

本專案為一個結合資料爬蟲、向量資料庫與 AI 模型的企業問答系統。

## 功能特色
- 自動爬取企業資料
- 建立向量資料庫
- AI 問答系統
- 圖片與資料清理工具
- Web 介面展示

## 系統架構
- Python
- ChromaDB
- Ollama / LLM
- GCP Compute Engine
- GitHub 版本管理

## 安裝方式

```bash
git clone https://github.com/1469173-sys/chatbot-project.git
cd chatbot-project
pip install -r requirements.txt
python app.py

使用環境

Python 3.9+
