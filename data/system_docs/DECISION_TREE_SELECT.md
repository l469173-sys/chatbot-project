# 設備選型決策樹

---

【第一層：測什麼？】

Q1：是否為雷射？
→ 是 → VCSEL

Q2：是否為消毒 / UVC？
→ 是 → SRI-4000UVC / HA-4000UVC

Q3：是否為植物燈？
→ 是 → SRI-PL-6000

Q4：是否為材料？
→ 是 → 進入材料樹

Q5：是否為照明？
→ 是 → 進入照明樹

---

【材料樹】

Q：測穿透？
→ 是 → SM-HR-7200 / SRI-2500TR

Q：測反射？
→ 是 → OOC-SM-2000 / SM-RF-8200

Q：測角度？
→ 是 → SM-DFG-450

---

【照明樹】

Q：測流明？
→ 是 → LM / SRI-2000LM

Q：測光強？
→ 是 → LI / LID / SRI-2000LID

Q：測配光？
→ 是 → LID-120 / 120S

Q：測色彩？
→ 是 → SRI / NE-3900

---

【顯示 / 面板】

Q：是否接觸式？
→ 是 → RL-5000

→ 否 → NE-2900 / NE-3900
